Latitude Project: Latitude vs. X

This project analyzes the relationship between the weather and distance from the equator. To complete this analysis I used the OpenWeather API to assemble data for over 500 cities. I assembled and cleaned the data using python and pandas and then created visualizations using Matplotlib. This website shows the source data and visualizations of weather vs. latitude created for this project. There is also an explantion of the trends and conclusions I observed from the data.
